# Test Data Directory

This directory contains test files for verifying agentic tool capabilities.

## Files:
- `sample_file.txt` - Plain text file for read testing
- `config.json` - JSON file for structured data reading
- `README.md` - This file (Markdown test)

## Usage:
These files are used by automated tests to verify that the model can:
1. Read files from the filesystem
2. Parse different file formats
3. Understand file contents and respond appropriately
