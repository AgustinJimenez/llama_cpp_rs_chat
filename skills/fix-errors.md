---
name: fix-errors
description: Find and fix compilation/runtime errors in a project
---

In the project at {{path}}:
1. Try to build/compile the project
2. Read the error messages carefully
3. For each error, find the source file and line
4. Fix the issue
5. Rebuild and verify the fix
6. Repeat until all errors are resolved

Do NOT skip errors or give up. Fix each one systematically.
